# aplicacao-grafica-clp
Repositório contendo implementação de fractal de Mandelbrot e interface, realizado como trabalho na disciplina de CLP.

### Alunos

- Augusto Ramos Molina | armolina@inf.ufpel.edu.br
- Leonardo Luís Müller | llmuller@inf.ufpel.edu.br

### Especificação do Trabalho

Implementar uma aplicação gráfica (sugestão: fractal de Mandelbrot, ray tracing, simulação de partículas) com apoio de duas linguagens de programação: Python e C (ou C++). O desafio consiste em realizar o uso conjunto de duas linguagens de programação, sendo o desafio especificamente, realizar o uso conjunto das linguagens. Python deve ser utilizado para oferecer uma interface com o usuário e apresentar a imagem gerada. A linguagem C (ou C++) deve ser utilizada para implementar o serviço de cálculo desejado.

### Pré-requisitos

* **GCC**

* **Python3**

    - Biblioteca Pillow

        `pip3 install Pillow`

### Utilização

* **1. Clone o repositório**

```sh
git clone https://github.com/gutormolina/aplicacao-grafica-clp.git
```

* **2. Construa o projeto**

```sh
~/aplicacao-grafica-clp$ make
```

* **3. Rode o projeto**

```sh
~/aplicacao-grafica-clp$ make run
```

### Forma de usar

A aplicação conta com diversos campos onde é possível escolher parâmetros para observação do fractal de Mandelbrot. Também há a possibilidade de acessar parâmetros recomendados, com áreas interessantes de observar.
